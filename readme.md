# Solución de Aplicación de Consola - Módulo 3 (Alkemy)

Este repositorio almacena la entrega correspondiente al proyecto integrador del **Módulo 3: Fundamentos de programación en JavaScript**, diseñado bajo directrices de Aprendizaje Basado en Proyectos (ABP).

## Arquitectura de Solución por Lecciones

La estructura de código implementada en el archivo `app.js` cumple minuciosamente con el programa técnico evaluativo:

1. **Lección #1: Introducción al Lenguaje:** Uso de flujos de salida con `console.log()`, entrada interactiva mediante `prompt()` y alertas modales con `alert()`.
2. **Lección #2: Variables y Condicionales:** Declaraciones consistentes con `let` y `const`, resolución de operaciones aritméticas básicas y desvíos lógicos `if-else` ante fallas de carga de datos.
3. **Lección #3: Arreglos y Ciclos:** Iteración estructurada de vectores numéricos mediante bucles `for` y `while`, acompañado de una función para el filtrado predictivo de elementos bajo una condición de frontera.
4. **Lección #4: Funciones:** Modularización de operaciones matemáticas de cálculo, parametrización de variables de entrada, uso de retornos de valor explícitos y anidación/anidamiento de llamadas entre funciones operativas.
5. **Lección #5: Conceptos de Objetos:** Implementación de colecciones estructuradas mediante objetos literales dotados de propiedades de datos y métodos funcionales internos. Procesamiento masivo de arreglos de objetos utilizando los métodos modernos `.forEach()` y `.map()`.

## Guía de Ejecución Local

1. Descargue el código fuente o clone el repositorio en su espacio de almacenamiento local.
2. Abra el archivo `index.html` en cualquier navegador web moderno (Chrome, Edge o Firefox).
3. Active el panel de control del programador presionando la tecla **F12** en Windows o **Cmd+Opt+I** en Mac.
4. Interactúe con las solicitudes dinámicas de las ventanas emergentes del navegador.
