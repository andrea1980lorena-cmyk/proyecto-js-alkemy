# Sistema Integrado de Control de Importaciones y Aduanas 🚢💼

## Descripción del Proyecto
Este sistema interactivo ha sido desarrollado en JavaScript nativo estructurado bajo los estándares de ECMAScript 6 (ES6+). Su propósito principal es resolver problemas complejos de la cadena de suministro internacional mediante programación modular, simulando el funcionamiento de un panel (*Dashboard*) logístico aduanero real para el control e internación de contenedores en puertos chilenos.

---
